Directory/Location Finder Help

The Directory/Location Finder page is used to obtain access to any location's directory in this system.

Choose any location to load its directory contents by tapping the right arrow icon next to any listing to load.

If your device's location settings allow this app to identify your current location, a list of local choices will be displayed automatically for you to choose from.

If no local listings are automatically displayed, or if you want to search a different area, then you may search for other locations/directories by entering a valid search value in the search bar.

Valid search values are:

  • -  Zip/Postal Code
  • -  Property/Location Name
  • -  Property/Location Address
  • -  Property/Location Code (may be provided to you by guest services staff)
